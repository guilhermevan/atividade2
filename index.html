Conhecendo o github

Entendendo os commits

  • Utilizar nomes de commits objetivos e curtos, até 50 caracteres
  • Os commits são "copias" de nosso projeto naquele momento
  • O campo obsevação irá ajudar futuramente a entender o motivo daquele commit, sempre utilize que achar necessario
  • Iniciar com letra maiuscula
  • Não terminar com ponto final
  • Não terminar de forma imperativa. Ex: Como se estivesse dando uma ordem. remover o banner

Dicas de boas praticas para nomes de commits

  • Adicionar:Funcionalidade da galeria de imagem adicionada
  • Alterar: Funiconalidade da galeria de imagens alterada
  • Deletar:Funcionalidade da galeria de imagens deletada
